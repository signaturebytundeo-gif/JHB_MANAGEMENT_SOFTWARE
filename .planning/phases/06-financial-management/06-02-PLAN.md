  <action>
Create `src/app/actions/financial-reports.ts` with these server actions (all 'use server', verify session):

1. `getRevenueByChannel(date: string)`: Get daily revenue for a given date. Query BOTH revenue sources (pitfall 2 — do NOT miss Order revenue):
   - `db.sale.groupBy({ by: ['channelId'], where: { saleDate: { gte: startOfDay, lte: endOfDay } }, _sum: { totalAmount: true } })`
   - `db.order.groupBy({ by: ['channelId'], where: { createdAt: { gte: startOfDay, lte: endOfDay }, status: { in: ['CONFIRMED', 'PROCESSING', 'SHIPPED', 'DELIVERED'] } }, _sum: { totalAmount: true } })`
   - Merge both into channel map: fetch `db.salesChannel.findMany({ select: { id: true, name: true } })`, combine Sale + Order revenue per channel.
   - Return: `{ channel: string; revenue: number }[]` — all Decimals converted via Number().

2. `getMonthlyRevenueTrend(year: number)`: Get monthly revenue for all 12 months of a year. For each month, sum Sale.totalAmount + Order.totalAmount (confirmed+ orders). Return `{ month: string; revenue: number; projection: number }[]` where projection uses benchmarks: Year 1 = $1.2M/12 = $100K/mo, Year 2 = $3.5M/12 = ~$291.7K/mo, Year 3 = $7.2M/12 = $600K/mo. Use year offset from 2026 as Year 1 to pick benchmark. If year is 2026, projection = 100000 per month.

3. `getBatchCOGS()`: Get COGS for all batches with materials. Query:
   ```
   db.batch.findMany({
     where: { status: { not: 'QUARANTINED' } },
     include: { materials: { include: { rawMaterial: true } }, product: { select: { name: true } } },
     orderBy: { productionDate: 'desc' },
   })
   ```
   For each batch compute: materialsCost = SUM(bm.quantityUsed * bm.rawMaterial.costPerUnit) — treat null costPerUnit as 0 and flag `costDataIncomplete: true`. laborCost = Number(batch.laborCostTotal ?? 0). overheadCost = Number(batch.overheadCostTotal ?? 0). totalCOGS = materials + labor + overhead. cogsPerUnit = totalCOGS / batch.totalUnits (guard div by 0).
   Return array of `{ batchId, batchCode, productName, totalUnits, materialsCost, laborCost, overheadCost, totalCOGS, cogsPerUnit, costDataIncomplete }`.

4. `getGrossMarginByProduct()`: Aggregate COGS data by product. For each product: average cogsPerUnit across batches. Query average sale price from Sale records (or PricingTier wholesale cash price as fallback). Gross margin = (avgSalePrice - avgCogsPerUnit) / avgSalePrice * 100. Return `{ productName: string; avgCOGS: number; avgSalePrice: number; grossMarginPct: number; batchCount: number }[]`.

All functions: use `Number()` on every Decimal field before returning (pitfall 3). Use date-fns for startOfDay/endOfDay/startOfMonth/endOfMonth/eachMonthOfInterval.
