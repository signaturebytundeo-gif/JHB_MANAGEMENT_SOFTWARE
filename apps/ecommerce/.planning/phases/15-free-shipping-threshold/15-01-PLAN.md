  <name>Task 1: Create FreeShippingBar component and add to root layout</name>
  <files>src/components/layout/FreeShippingBar.tsx, src/app/layout.tsx</files>
  <action>
Create `src/components/layout/FreeShippingBar.tsx` as a 'use client' component that:

1. Imports `useCartStore` from `@/lib/cart-store` and `formatPrice` from `@/lib/utils`.
2. Calculates `subtotal` from cart items: `items.reduce((sum, item) => sum + (item.isFreeSample ? 0 : item.price * item.quantity), 0)` — exclude free samples from the total (matches checkout route logic on line 46-49).
3. Defines `FREE_SHIPPING_THRESHOLD = 5000` (in cents, matching checkout route).
4. Computes `remaining = FREE_SHIPPING_THRESHOLD - subtotal` and `progress = Math.min((subtotal / FREE_SHIPPING_THRESHOLD) * 100, 100)`.
5. Renders a thin horizontal bar (full width, fixed position or static at top of page) with:
   - Background: `bg-brand-gold/10` with gold accent text
   - When cart is empty or subtotal is 0: Static text "Free shipping on orders $50+" with a small truck/shipping icon (inline SVG)
   - When subtotal > 0 but < $50: "Add {formatPrice(remaining)} more for FREE shipping!" with a progress bar beneath showing visual progress (gold fill on dark track)
   - When subtotal >= $50: "You qualify for FREE shipping!" with a checkmark icon, text in brand-gold
   - Height: compact (py-1.5 or py-2), text-xs sm:text-sm, centered text
   - Must include `useEffect` with `useCartStore.persist.rehydrate()` to handle SSR hydration (same pattern as CartDrawer)

6. In `src/app/layout.tsx`:
   - Import `FreeShippingBar` from `@/components/layout/FreeShippingBar`
   - Render `<FreeShippingBar />` ABOVE `<Navigation />` inside the flex column div, so it appears as the topmost element on every page
   - The component order should be: FreeShippingBar, Navigation, main, Footer
  </action>
  <verify>Run `npx next build` (or `npm run build`) to confirm no TypeScript or build errors. Visually confirm in dev server that the bar appears above the navigation on the homepage, shop page, and a product detail page.</verify>
