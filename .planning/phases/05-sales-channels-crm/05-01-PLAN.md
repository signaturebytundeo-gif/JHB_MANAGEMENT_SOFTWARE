   - updateLeadSchema: all fields optional plus id required

4. Create pricing.ts validator:
   - createPromotionalPricingSchema: productId (min 1), name (min 1), discountPercent optional (coerce number, min 0, max 100), fixedPrice optional (coerce number, min 0), startDate (coerce date), endDate (coerce date)
   - Add .refine() to ensure either discountPercent OR fixedPrice is provided (not both, not neither)

5. Create pricing.ts utility (src/lib/utils/pricing.ts):
   - Export type PriceCalculation = { basePrice: number; discountPercent: number; discountReason: string | null; finalPrice: number }
   - Export async function calculateLineItemPrice(productId: string, tierName: string, unitQuantity: number, frequencyDiscount: 'quarterly' | 'annual' | null, tx: PrismaClient): Promise<PriceCalculation>
   - Implementation:
     a. Fetch base price from PricingTier where productId and tierName match. If not found, throw error.
     b. Fetch product to get unitsPerCase. Calculate caseQuantity = Math.floor(unitQuantity / unitsPerCase). If unitsPerCase is null, skip volume discount.
     c. Check PromotionalPricing where productId matches, isActive=true, startDate <= now, endDate >= now. If found, use its discountPercent (or calculate from fixedPrice). Set discountReason = "promotional".
     d. If no promotional: look up VolumeDiscount where minCases <= caseQuantity AND (maxCases >= caseQuantity OR maxCases is null). Get volumePercent.
     e. If no promotional: look up FrequencyDiscount where frequency matches frequencyDiscount param. Get freqPercent.
     f. Apply Math.max(volumePercent, freqPercent) as the discount. Set discountReason accordingly ("volume" or "frequency").
     g. Calculate finalPrice = basePrice * (1 - discountPercent/100).
     h. Return { basePrice, discountPercent, discountReason, finalPrice }.
   - CRITICAL: Do NOT stack volume + frequency discounts. Only the higher of the two applies. Promotional pricing supersedes both.
   - Use Number() conversion at Decimal boundaries (Prisma Decimal -> JS number).
