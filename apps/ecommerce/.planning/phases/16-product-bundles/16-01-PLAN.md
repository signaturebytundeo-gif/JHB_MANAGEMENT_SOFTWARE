  <files>src/types/product.ts, src/data/bundles.ts, src/data/products.ts</files>
  <action>
1. Extend `src/types/product.ts` — add a `Bundle` interface:
   ```
   export interface Bundle {
     id: string
     name: string
     description: string
     slug: string
     price: number          // bundle price in cents (lower than sum of included items)
     image: string
     images: string[]
     includedProductIds: string[]   // references to Product.id values
     savings: number        // amount saved in cents vs buying individually
     callouts: string[]
     inStock: boolean
     rating: number
   }
   ```
   Keep the existing Product interface unchanged. Do NOT merge Bundle into Product — they are separate types with different semantics (Bundle has includedProductIds and savings, Product does not).

2. Create `src/data/bundles.ts` with 3 pre-defined bundles:
   - **Starter Bundle** (id: 'starter-bundle'): includes jerk-sauce-2oz ($6.99) + jerk-sauce-5oz ($11.99) + escovitch-pikliz-12oz ($11.99). Individual sum = $30.97. Bundle price = $24.99 ($5.98 savings). Image: '/images/products/product-lineup.jpg'.
   - **Heat Pack** (id: 'heat-pack'): includes jerk-sauce-5oz ($11.99) + jerk-sauce-10oz ($19.99) + escovitch-pikliz-12oz ($11.99). Individual sum = $43.97. Bundle price = $36.99 ($6.98 savings). Image: '/images/products/product-lineup.jpg'.
   - **Gift Set** (id: 'gift-set'): includes all 4 products: jerk-sauce-2oz + jerk-sauce-5oz + jerk-sauce-10oz + escovitch-pikliz-12oz. Individual sum = $50.96. Bundle price = $42.99 ($7.97 savings). Image: '/images/products/product-lineup.jpg'.
   Export: `bundles` array, `getBundleBySlug(slug)`, `getBundleById(id)` helper functions.
