1. Create crm-emails.ts following the existing customer-emails.ts pattern:
   - Import Resend from 'resend'
   - sendRenewalReminderEmail({ customerEmail, customerFirstName, planName, renewalDate }): sends inline HTML email via Resend with JHB branding, subject "Your JHB Subscription Renewal is Coming Up", body mentions renewal date and plan name. Return { success: boolean; error?: string }.

2. Create crm-subscriptions.ts server actions:
   - getSubscriptionMembers(): findMany including customer (firstName, lastName, email), plan (name, billingCycle, price), orderBy createdAt desc. Also compute loyaltyEligible boolean: differenceInMonths(now, startDate) >= 6 AND loyaltyRewardAt is null.
   - createSubscriptionMember(prevState, formData): validate with createSubscriptionMemberSchema, db.subscriptionMember.create, revalidatePath('/dashboard/customers'), return { success: true }.
   - updateSubscriptionStatus(prevState, formData): validate with updateSubscriptionStatusSchema, update status. If CANCELLED, set cancelledAt = now(). revalidatePath, return { success: true }.
   - sendRenewalReminders(): query SubscriptionMember where status=ACTIVE AND renewalDate <= addDays(now, 30) AND renewalDate > now AND renewalReminderSentAt IS NULL. Include customer relation for email/name. For each result, call sendRenewalReminderEmail, then update renewalReminderSentAt = now() inside db.$transaction (one per member for atomicity). Return { sent: number, errors: number }.
   - awardLoyaltyReward(memberId: string): update loyaltyRewardAt = now() on SubscriptionMember. This is a manual trigger. revalidatePath, return { success: true }.

3. Create SubscriptionMemberForm.tsx ('use client'):
   - Props: { customers: { id, firstName, lastName }[]; plans: { id, name, billingCycle, price }[]; onClose: () => void }
   - useActionState with createSubscriptionMember
   - Fields: customerId (Select dropdown), planId (Select dropdown), startDate (date input), renewalDate (date input, optional), notes (textarea)
   - Modal overlay pattern (fixed inset-0)
   - useEffect for toast.success + onClose on success

4. Create SubscriptionMemberList.tsx ('use client'):
   - Props: { members: SubscriptionMember[] with customer and plan relations; customers: Customer[]; plans: SubscriptionPlan[] }
   - "Add Subscription" button that opens SubscriptionMemberForm modal
   - "Send Renewal Reminders" button that calls sendRenewalReminders server action directly (not form — use useTransition), show toast with sent/error count
   - Table/cards with columns: Customer Name, Plan, Status (badge with color: ACTIVE=green, PAUSED=yellow, CANCELLED=red, EXPIRED=gray), Start Date, Renewal Date, Loyalty (months count + "Eligible" badge if >= 6 months and no reward yet)
   - Status change dropdown per row: Pause, Cancel buttons that call updateSubscriptionStatus
   - "Award Reward" button visible when loyaltyEligible and loyaltyRewardAt is null
   - Client-side filter by status
   - Responsive: desktop table, mobile cards (per established pattern)
  </action>
